## Locked decisions

1. **Tabbed workspace.** A change opens in its OWN tab — a top bar with the product
   switcher + a tab strip (a Board tab + each open change as a closable tab, each with
   a stage-coloured dot). Replaces the single-screen Board↔Thread swap. Several changes
   open at once, like an editor / Linear.
2. **Change-scoped custom left nav.** Inside an open change: the change's name, its
   stage track (Recon → Ship), and view switches — **Conversation · Files · Brain ·
   Preview**. This REPLACES the right-hand rail — one navigation model, not two.
3. **Turn-Cards chat.** Each agent turn = one card: a **2–3 sentence summary** shown by
   default, with TWO independent expanders — **"show the full reply"** (the agent's
   complete prose for the turn) and **"N steps"** (the recorded steps). The founder
   scrolls summaries, not raw steps; both expanders collapsed by default. A sticky
   stage/status bar sits at the top of the conversation. (Amended 2026-06-04: summary
   is 2–3 sentences + a full-reply expander, not a one-line headline — founder request.)
4. **Per-turn summary.** The card summary is generated by a short, cached, headless
   `claude -p` pass per agent turn that MUST use **Haiku** (`claude -p --model
   claude-haiku-…`) — the small, fast, cheap tier; never a larger model for this. It
   returns a **2–3 sentence** plain-English summary (not a one-liner). Keyed by the
   turn's content so it is computed ONCE and never regenerated, produced off the
   critical path (non-blocking) with a **first-few-sentences fallback** while pending.
   The step list comes straight from the recorded actions (the transcript's
   tool-use/result records) — no model needed.
5. **Composer.** Floating-inset (lifted off the bottom edge, rounded) with a
   "working · pause · stop" status chip directly above it — both the live action and
   the input sit in the same comfortable reach (the founder's "hard to look to the
   bottom" fix). The box is **bigger by default and auto-grows**, with a **Slack-style
   expand control** that pops it to a tall draft editor for longer messages. (Amended
   2026-06-04 — founder request.)
6. **Heroicons** throughout (MIT, Tailwind Labs): outline 24 for nav/views, mini 20 for
   inline step rows; `currentColor` so they inherit the token colours.

## Principles honoured

- **AI-01** — chat is the coordination layer; the agent's work renders as scannable
  structure (cards + folded steps), not a wall of message text.
- **AI-07** — steps stay attributed and one tap away; the headline is an honest
  summary, no false-certainty spectacle.
- **CL-02** — progressive disclosure: steps collapsed by default.
